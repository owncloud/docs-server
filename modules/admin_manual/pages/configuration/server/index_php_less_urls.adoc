= Enable index.php-less URLs
:toc: right

== Introduction

Note that with the Docker Image provided by ownCloud, `index.php` less URLs are enabled by default.

== Check

You can check this by printing the `config/config.php` file via the bind mount after the first setup which will contain the following keys. Note that the URL matches your setup:

[source,php]
----
'overwrite.cli.url' => 'https://example.com',
'htaccess.RewriteBase' => '/',
----
